# 🚀 Sakshi Gupta Portfolio

A modern, responsive, and professional portfolio website built with **React, TypeScript, Vite, Tailwind CSS, and Framer Motion**. This portfolio showcases professional experience, technical expertise, achievements, and project highlights as a Frontend Lead Engineer specializing in scalable UI architecture, React, Angular, and enterprise-grade web applications.
